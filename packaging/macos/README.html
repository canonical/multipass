A couple details:

  • Use a keyboard shortcut (⌘⌥U) to open a shell in your primary instance
  • Your home directory will be automatically mounted in
    ~/Home in the primary instance
  • There's a checkbox under the "About" item in the system menu to toggle the system menu starting automatically on login

Make sure to check out the latest release notes on GitHub.

Minimum System Requirements

  • macOS 13 Ventura or later.
